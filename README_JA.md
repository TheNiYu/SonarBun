# 🎧 SonarBun - by TheNiYu

<div align="center">

[English](README.md) | **日本語**

</div>

Windows向けのリアルタイム音声文字起こし・翻訳ツールです。

SonarBunは、デスクトップや出力デバイスの音声、またはマイクなどの入力デバイスを聞き取り、Whisperでローカル文字起こしを行います。その後、オンラインAPIサービスまたはローカルのオフラインモデルを使用して翻訳できます。

---

## ⚙️ 主な機能

- 🎧 Windowsの出力音声やループバック音声を取得
- 🎙️ マイク、オーディオインターフェース、仮想入力に対応
- 📝 faster-whisperによるローカル音声文字起こし
- 🌐 ユーザー自身のAPIキーを使ったオンライン翻訳
- 📴 モデルのダウンロード後に利用できるローカルオフライン翻訳
- 🌍 元の言語と翻訳先の言語を選択可能
- 📚 編集可能な用語集プロファイルと翻訳キャッシュ
- 📄 最新の翻訳を`live.txt`へ出力
- 🖥️ 英語と日本語のGUI表示
- ✅ インストール不要: ZIPを展開して`.exe`を起動するだけ

---

## 🚀 使い方

1. [Releases](../../releases)から`SonarBun-v1.0.0-win64.zip`をダウンロードします
2. ZIP全体を通常のフォルダへ展開します
3. `SonarBunLiveTranslator.exe`を起動します
4. 次の項目を選択します:
   - 出力/ブラウザ音声、または入力/マイク
   - 音声デバイス
   - 元の言語
   - 翻訳先の言語
5. **Settings**でOnlineまたはOffline翻訳を選びます
6. **Start**を押します

ZIPファイルの中から直接実行しないでください。`_internal`フォルダと`glossaries`フォルダは、実行ファイルと同じ場所に残す必要があります。

詳しいガイドは[イタリア語](docs/Guida_Utente_SonarBun_IT.pdf)、[英語](docs/SonarBun_User_Guide_EN.pdf)、[日本語](docs/SonarBun_User_Guide_JA.pdf)で読むことができます。

## 🌐 オンラインモード

オンラインモードでは、ユーザーが用意した個人用APIキーを使用します。SonarBunにAPIキーは付属していません。

APIキーはWindows DPAPIを使ってローカルで暗号化されます。オンライン翻訳時には、ローカルで文字起こしされたテキストが選択したAPIプロバイダーへ送信されます。SonarBunは翻訳プロバイダーへ生の音声を送信しません。

利用可能なサービス、料金、使用量制限、データの取り扱いは、それぞれの外部サービスによって管理されます。

---

## 📴 オフラインモード

オフラインモードではAPIサービスを必要としません。インターネット接続が必要なのは、選択した翻訳モデルを初回にダウンロードするときだけです。

現在利用できるオフラインモデルは約630 MBです。このモデルは独自のライセンスに基づき、個人かつ非商用の利用を目的としています。オフライン翻訳はオンライン翻訳より文脈理解が弱い場合があります。

---

## 💻 動作要件

- Windows 10またはWindows 11、64-bit
- 正常に動作する音声入力/出力デバイス
- モデルのダウンロードおよびオンラインモード用のインターネット接続
- NVIDIA CUDA対応GPUは任意

CPUモードではNVIDIAソフトウェアは必要ありません。CUDAを使用する場合は、対応するNVIDIAハードウェア、ドライバー、およびユーザーガイドに記載されたランタイムコンポーネントが必要です。

## 🔐 プライバシー

SonarBunには、分析、テレメトリー、広告、ユーザーアカウントは含まれていません。

設定、履歴、キャッシュ、モデル、出力ファイルはローカルに作成されます。詳しくは[PRIVACY.md](PRIVACY.md)をご覧ください。
